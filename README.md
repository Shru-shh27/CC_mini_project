# 🔢 Number Theory Calculator

A simple web-based project to perform common **number theory operations** used in competitive programming.

---

## 📌 Features

- GCD (Greatest Common Divisor)
- LCM (Least Common Multiple)
- Prime Number Check
- Modular Arithmetic:
  - (a + b) mod m
  - (a × b) mod m
  - Fast Exponentiation (a^b mod m)

---

## 🧠 Concepts Used

- Euclidean Algorithm (GCD)
- LCM using GCD
- Prime checking (√n method)
- Modular Arithmetic
- Binary Exponentiation
